{
-------对父子元素都只设置width的情况下：
相互之间没有影响。
}

{
---------对子元素设定min或max 父元素只设置width
min-width：
	如果父元素宽度小于最小宽度，不会继承父元素宽度。一直显示为最小宽度。
	如果父元素宽度大于最小宽度，会继承父元素宽度。一直显示为父元素宽度。

max-width:
	如果父元素宽度小于最大宽度，会继承父元素宽度。一直显示为父元素宽度。
	如果父元素宽度大于最大宽度，不会继承父元素宽度。一直显示为最大宽度。

对子元素同时设定min和max
	如果父元素宽度在最小和最大之间。一直显示为父元素宽度。
	如果父元素宽度大于最大宽度。一直显示为最大宽度。
	如果父元素宽度小于最小宽度。一直显示为最小宽度。
}


{
--------对父元素设定min和max 子元素只设置width
	子父元素之间大小互不影响。
}

{
--------对父元素子元素同时设置min和max
	父元素的显示不受影响。
	
	
父元素最大宽度最大的情况下：子元素的显示：1如果父元素最小宽度大于子元素最大宽度{
								  		子元素一直显示自己的最大宽度				
                                                                 		 }
		      			  2如果父元素最小宽度在子元素两个宽度之间{
										子元素显示在父元素最小宽度和子元素最大宽度之间
										}	
		    			  3如果父元素最小宽度小于子元素最小宽度{
										子元素显示在自己的最大宽度和最小宽度之间
										}
父元素最大宽度在子元素最大和最小之间的情况：
					  2如果父元素最小宽度在子元素两个宽度之间{
										子元素显示在父元素最小宽度和父元素最大宽度之间
                                                                                }
 					  3如果父元素最小宽度小于子元素最小宽度{
										子元素显示在自己的最小宽度和父元素最大宽度之间
										}
父元素最大宽度小于子元素最小宽度的情况：
					  3如果父元素最小宽度小于子元素最小宽度{
										子元素一直显示自己的最小宽度
										}
  













		
}
